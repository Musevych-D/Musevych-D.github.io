CSS animations and display:flex css property

Task is divided into few subtasks:
Each task needs a section containing 4 div's. Width of div's should be such to fir into their thection (in a row). Section must have display:flex property. Each section should contain a header with current property, that is animated right now.

Subtask #1:
Section #1 should have an animation of changing of all variants of JUSTIFY-CONTENT property one by one.

Subtask #2:
Section #2 should have an animation of changing of all variants of ALIGN-ITEMS property one by one.

Subtask #3:
Section #3 should have an animation of changing of all variants of FLEX-DIRECTION property one by one.

Subtask #4:
Section #4 should have an animation of changing width of div's with section flex-wrap:wrap property until they reach maximum width of a row.

Subtask #5:
Section #5 should have an animation of ALIGN-CONTENT after wrap has occurred.

Subtask #6*:
Section #6 should contain header with 4 div's. All animations from tasks 1-5 should be presented here. Header should show a name of a current animation runned.
